## Introduction

Compared to Android (Kotlin) or iOS (Swift), Flutter is easier to learn and beginner-friendly. If you are new to programming and interested in mobile app development, Flutter is a great place to start.  

By the end of this guide, you should be able to:

- Create a Flutter project  
- Understand basic widgets  
- Build a simple UI  

---

## Creating a New Flutter Project

Here are the basic steps:

1. **Install IDE**: Android Studio or VSCode. (We’ll use Android Studio in this guide.)  
2. **Install Flutter**: Follow [official Flutter installation](https://flutter.dev/docs/get-started/install).  
3. **Install Plugins**: Flutter & Dart plugins in your IDE.  
4. **Create a New Project**: File → New → Flutter Project → Enter project details.
